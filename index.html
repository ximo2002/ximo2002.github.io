我的作品

  • Ollama 是一个轻量级、可扩展的框架，用于在本地机器上构建和运行大型语言模型。它提供了创建、运行和管理模型的简单API，以及一个预构建模型库，这些模型可以轻松地应用于多种应用程序。Ollama 支持多种模型，如 Llama 3.1、Phi 3 Mini/Medium、Gemma 2 等，并且提供了详细的安装和使用指南，包括 Docker 镜像、CLI 命令、REST API 以及社区集成

  • Qwen2Audio是QwenAudio的下一代版本，它支持音频和文本输入并生成文本输出。该模型具备语音聊天、音频分析功能，并支持超过8种语言和方言。Qwen2Audio在多个基准数据集上超越了先前模型，并且已经在Hugging Face和ModelScope上开源。此外，提供了使用示例和未来计划。

  • DDColor是一个图像着色工具由阿里巴巴集团开发，用于实现逼真的图像着色效果。
